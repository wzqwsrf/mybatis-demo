## 关于mybatis-demo
---------------------------------------
1. mybatis单表查询示例。
2. mybatis关联查询示例。
3. 不包含复杂查询。

## 运行
---------------------------------------
1. 建表，sql为mybatis-demo/src/main/resources/mysql
2. git clone git@github.com:wzqwsrf/mybatis-demo.git
3. cd mybatis-demo
4. mvn clean install 
5. 在idea或者eclipse中进行单个测试。
6. mvn test也可以，有可能因为代码先后关联关系出错。
